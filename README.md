# proyectoTiendaOrdenadoresJava
Este proyecto es una tienda online de ordenadores desarrollada con Spring Boot y Java 21. Permite gestionar productos, realizar operaciones CRUD sobre la base de datos y mostrar la información en vistas Thymeleaf. Se ha diseñado para ser escalable y fácil de mantener, integrando tanto una base de datos embebida H2 para pruebas como MySQL.
